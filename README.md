# Country Explorer

A clean React + TypeScript country search app that uses the REST Countries API.

Features
- Search countries by name
- Filter by region
- Show flag, population, capital, and region
- Typed form handling with validation using `react-hook-form` and `zod`
- API wrapper for REST Countries requests

## Setup

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm run dev
```

Open the local URL shown in the terminal.
